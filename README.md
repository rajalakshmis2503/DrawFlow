# DrawFlow

Traditional drawing applications rely on devices such as a mouse, stylus, or touchscreen for user interaction. DrawFlow offers a more intuitive and interactive approach by allowing users to draw and write in the air using hand gestures captured through a webcam. The project utilizes computer vision techniques to track movement in real time and translate gestures into digital strokes on a virtual canvas.

Designed for artists, educators, students, and presenters, DrawFlow provides a touchless drawing experience that enhances creativity and engagement. The application includes features such as freehand drawing, color selection, erasing tools, and virtual whiteboard functionality. By combining real-time motion tracking with an easy-to-use interface, DrawFlow creates a seamless bridge between physical gestures and digital interaction, making drawing and visual communication more accessible and engaging.
