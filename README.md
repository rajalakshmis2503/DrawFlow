# DrawFlow
Hand gesture recognition is a fundamental component of human-computer interaction, enabling intuitive control and communication in various applications. However, existing systems often face challenges in accurately detecting and interpreting hand gestures in real-time, leading to limited usability and performance issues. This project proposes an approach to hand gesture recognition leveraging advanced computer vision techniques and machine learning algorithms. By integrating background subtraction, contour detection, and gesture classification algorithms, the proposed system aims to achieve robust and real-time recognition of hand gestures. Targeted users include artists, educators, presenters, and individuals with disabilities, who can benefit from the system's intuitive interface and versatile functionality. The output of the system includes real-time visualization of detected hand gestures and corresponding actions, empowering users with enhanced control and interaction capabilities in digital environments.

DrawFlow is an innovative application that combines real-time interaction with advanced text recognition capabilities. It utilizes a webcam interface to allow users to draw directly on the screen using hand gestures or physical objects. Once the user creates their drawings or writes characters in the air, the application employs EasyOCR to recognize and interpret the drawn characters or text. This seamless integration enables intuitive and interactive drawing sessions, where recognized text can be further processed or converted into digital formats. Ideal for educational purposes, creative projects, and interactive presentations, DrawFlow leverages modern technology to bring a dynamic and engaging user experience.
